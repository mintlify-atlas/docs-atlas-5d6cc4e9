### Ubuntu 22.04 Official Release

<Info>
  **Release Date:** Recent
  
  **Platform:** Linux (Ubuntu 22.04)

**What's New:**

<AccordionGroup>
  <Accordion title="Official Ubuntu Support">
    **Platform Compatibility:**
    - ✅ Ubuntu 22.04 Desktop (kernel 6.8.0-52-generic)
    - Full adaptation and verification completed
    - AppImage distribution for easy installation
    
    **Installation:**
    - Download `Duix.Avatar-x.x.x.AppImage` from [GitHub Releases](https://github.com/duixcom/Duix.Avatar/releases)
    - Make executable: `chmod +x Duix.Avatar-x.x.x.AppImage`
    - Run: `./Duix.Avatar-x.x.x.AppImage`
    - For root users: `./Duix.Avatar-x.x.x.AppImage --no-sandbox`
    
    <Warning>
      Other Linux distributions have not been tested yet. Ubuntu 22.04 is the officially supported platform.
    </Warning>
  </Accordion>

<Tabs>
  <Tab title="Windows">
    **Supported Versions:**
    - Windows 10 (19042.1526 or higher)
    - Windows 11
    
    **Installation:**
    - Download `Duix.Avatar-x.x.x-setup.exe`
    - Double-click to install

    - Kernel 6.8.0-52-generic
    
    **Installation:**
    - Download `Duix.Avatar-x.x.x.AppImage`
    - No installation required
    - Run directly from downloads
    
    **Requirements:**
    - 100GB+ free space
    - NVIDIA GPU with drivers
  </Tab>
</Tabs>

### Docker Services

<AccordionGroup>
  <Accordion title="Full Version (3 Services)">
    Complete offline operation:
    
    1. **Duix.Avatar-asr** - Automatic Speech Recognition
       - Based on fun-asr technology
       - Speech-to-text conversion
       - Audio analysis
    
    2. **Duix.Avatar-tts** - Text-to-Speech
       - Based on fish-speech-ziming
       - Voice cloning
       - Multi-language synthesis
    
    3. **Duix.Avatar-gen-video** - Video Generation
       - Avatar animation
       - Lip-sync processing
       - Video synthesis
    
    **Download Size:** ~70GB
    **Installation:** `docker-compose up -d`
  </Accordion>

  <Accordion title="Lite Version (1 Service)">
    Streamlined deployment:
    
    - **Duix.Avatar-gen-video** only
    - Requires external TTS/ASR
    - Smaller download size
    - Lower resource requirements
    
    **Installation:** `docker-compose -f docker-compose-lite.yml up -d`
  </Accordion>
</AccordionGroup>
